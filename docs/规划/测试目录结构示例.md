## 六、边界场景（分类行为需验证）

### 6-A 图片略低于 85% 阈值（边界 mixed/photo）

> 15 张图片 + 3 个视频 → 图片比 83.3% < 85% → 预期 `mixed`

```
source/
└── 婚礼花絮/
    ├── photo_001.jpg … photo_015.jpg   (15张)
    ├── highlight_1.mp4
    ├── highlight_2.mp4
```

## 分类规则速查表

| 条件 | 分类 |
|------|------|
| 文件夹名含 `漫画` / `manga` / `comic` | `manga` |
| 含 `.cbz` / `.cbr` / `.cb7` / `.cbt` 文件 | `manga` |
| 图片比例 ≥ 85% | `photo` |
| 视频比例 ≥ 85% | `video` |
| 图片比例 ≥ 15% **且** 视频比例 ≥ 15% | `mixed` |
| 无图片/视频文件 | `other` |

> 注：分类比例计算仅基于直接子文件的扩展名，不递归统计子目录。
